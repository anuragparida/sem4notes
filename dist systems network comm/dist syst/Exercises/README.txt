This Eclipse project contains selected sample solutions for implementation tasks of
the textbook "Middleware in Java" published by SpringerNature. The book is available
as print- and online-version via SpringerLink: 
https://link.springer.com/book/10.1007/978-3-322-80262-0

The sample solutions are organized in packages named after the chapters and exercises in
the book. Use the sample solutions as the starting point for your own studies.
